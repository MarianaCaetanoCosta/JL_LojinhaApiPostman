## Postman
1. Acesse o site [Postman](https://www.postman.com/)
2. Faça o cadastro ou acesse sua conta
3. Crie um Workspace "PTQS-JulioDeLima"
4. Importe a coleção "Loginha Api.postman_collection.json"
